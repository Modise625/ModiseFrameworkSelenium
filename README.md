# Selenium java Framework with TestNG and Maven
This is a Selenium Java Framework that uses TestNG and Maven for testing web applications. The framework is designed to be easy to use and maintain, with a focus on readability and reusability.

## Prerequisites
- Java Development Kit (JDK) 8 or higher
- Maven 3.6 or higher
- An IDE such as IntelliJ IDEA or Eclipse
- WebDriver for the browser you want to test (e.g., ChromeDriver for Chrome)
